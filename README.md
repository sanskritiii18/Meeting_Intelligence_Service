# Meeting Intelligence Service

## Features

- JWT Authentication
- Meeting Management
- AI Meeting Analysis (Gemini)
- Grounded Citations
- Action Item Management
- Overdue Detection
- Reminder Scheduler
- Discord Integration
- Trace IDs
- Structured Logging
- Swagger Documentation

## Tech Stack

- Flask
- PostgreSQL
- SQLAlchemy
- Gemini API
- APScheduler
- Marshmallow
- Flasgger

## Setup

1. Clone repo
2. Create .env
3. Install dependencies

pip install -r requirements.txt

4. Run

python main.py
